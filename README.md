# Thrisen

Interactive split-screen website built with Vite + React, React Router, React Three Fiber, and GSAP.

## Tech Stack

- Vite + React (SPA)
- React Router
- React Three Fiber + Drei + Three.js
- GSAP
- react-helmet-async for per-route SEO metadata

## Run Locally

```bash
npm install
npm run dev
```

App runs on [http://localhost:5173](http://localhost:5173) by default.

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Production build
- `npm run preview` - Preview production build
- `npm run typecheck` - TypeScript checks
- `npm run lint` - ESLint checks

## Routes

- `/` - Bridge (split-screen landing + 3D scene)
- `/personal` - Personal world
- `/professional` - Professional world
